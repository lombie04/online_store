# Online Store (PHP)

A role-based online store system built with PHP.

## Features
- Admin dashboard (manage users, retailers, products)
- Retailer panel (add/manage products, orders)
- Customer storefront (browse, cart, checkout)
- Staff order handling
- Authentication and role-based access

## Tech Stack
- PHP
- MySQL
- HTML/CSS
- XAMPP (local dev)

## Notes
This is a student project created for learning purposes.
